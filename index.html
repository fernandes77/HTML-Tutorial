<html>

The <html> tag wraps all of the content on the page. It is also known as the root element.

<head>

The <head> tag contains everything that the webpage will not show. This includes keywords, page description, css references, etc.

<meta>

The <meta> tag adds metadata (such as the character set) to the document.

<link>

The <link> tag specifies relationships between the current document and an external resource. Use the href attribute to point to a file and the rel attribute to specify what the file does.

<script>

The <script> tag loads a JavaScript file.

<title>

The <title> tag specifies the webpage's title.

<body>

The <body> contains all of the content that is displayed to the user.

<h1>, <h2>, <h3>, <h4>, <h5>, <h6>

These tags are the headings of the page. Their font sizes are:
<h1>: 2em, <h2>: 1.5em, <h3>: 1.17em, <h4>: 1em, <h5>: 0.83em, <h6>: 0.75em.

<p>

The <p> tag makes your text a paragraph.

<em>

The <em> tag gives emphasis to your text, like this.

<strong>

The <strong> tag gives importance to your text, like this.

<img>

The <img> tag displays an image. You can specify the source with the src attribute and if the image fails to load, an alternative text with the alt attribute.

<a>

The <a> tag makes your text a link, like this one. You can specify where the link leads to with the href tag. You can also specify the text that is shown when hovering over it, with the title attribute, and the browsing context (if it opens a new window, tab, etc) with the target attribute.

<span>

The <span> tag wraps content that you want to style with CSS, without giving any extra meaning to it.

<ul>, <li>

The <ul> tag makes an unordered list, of which each item is wrapped with a <li> tag.

<ol>, <li>

The <ol> tag makes an ordered list, of which each item is wrapped with a <li> tag.

<dl>, <dt>, <dd>

These tags mean, respectively: description list, description term and description definition. These are used in lists of terms and their definitions or in questions and answers.

melancholic
Melancholic individuals tend to be analytical and detail-oriented, and they are deep thinkers and feelers.
phlegmatic
Phlegmatic individuals tend to be relaxed, peaceful, quiet, and easy-going.
sanguine
Sanguine personality type is described primarily as being highly talkative, enthusiastic, active, and social.
choleric
Choleric individuals tend to be more extroverted. They are described as independent, decisive, goal-oriented, and ambitious.

<blockquote>

If a section of block level content (be it a paragraph, multiple paragraphs, a list, etc.) is quoted fromsomewhere else, you should wrap it inside a <blockquote> element to signify this, and include a URL pointing to the source of the quote inside a cite attribute.
