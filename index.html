This is just text. No tags.

<html>

The <html> tag wraps all of the content on the page. It is also known as the root element.

<head>

The <head> tag contains everything that the webpage will not show. This includes keywords, page description, css references, etc.

<meta>

The tag <meta> adds metadata (such as the character set) to the document.

<title>

The <title> tag specifies the webpage's title.

<body>

The <body> contains all of the content that is displayed to the user.

<h1>, <h2>, <h3>, <h4>, <h5>, <h6>

These tags are the headings of the page. Their font sizes are:
<h1>: 2em, <h2>: 1.5em, <h3>: 1.17em, <h4>: 1em, <h5>: 0.83em, <h6>: 0.75em.

<p>

The <p> tag makes your text a paragraph.

<em>

The <em> tag converts your font to italic, like this.

<strong>

The <strong> tag converts your font to bold, like this.

<img>

The <img> tag displays an image. You can specify the source with the src attribute and if the image fails to load, an alternative text with the alt attribute.

<a>

The <a> tag makes your text a link, like this one. You can specify where the link leads to with the href tag. You can also specify the text that is shown when hovering over it, with the title attribute, and the browsing context (if it opens a new window, tab, etc) with the target attribute.
